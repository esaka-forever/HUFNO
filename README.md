The code is used for hybrid U-Net and Fourier neural operator (HUFNO) model developed in the paper: https://arxiv.org/abs/2504.13126. The work is built upon the pioneering works by Li et al. (https://doi.org/10.48550/arXiv.2010.08895), Tran et al. (https://arxiv.org/abs/2111.13802), and Li et al. (https://doi.org/10.1063/5.0158830).
